# PedeSim

Sistema para campanhas de arrecadação do Clube de Desbravadores.

## Objetivo
Gerenciar pedidos de pizza, sabores e controle financeiro via PIX.

## Funcionalidades
- Campanhas por período
- Sabores por campanha
- Pedido online
- Controle de pagamento
- Organização da produção

## Tecnologias
- Next.js
- Supabase
- PostgreSQL

## Execução local

Crie um arquivo `.env.local`:

```env
NEXT_PUBLIC_SUPABASE_URL=...
NEXT_PUBLIC_SUPABASE_ANON_KEY=...
